## Usage

### 1. Clone the project & the dependencies (via git submodules)
```shell
git clone https://github.com/atomicframeworks/ccip-bootcamp-day-2-homework.git --recursive
```

### 2. CD in to the directory

```shell
cd ccip-bootcamp-day-2-homework/
```

### 3. Build

```shell
forge build
```

### 4. Test

```shell
forge test
```
